1. 获取帮助

```cmd
命令 /？
```

2. 进入磁盘

```cmd
D: #进入d盘
F: #进入F盘
```

3. cd命令

```cmd
cd /? #获取使用帮助
cd \ #跳转到硬盘的根目录
cd C:\WINDOWS #跳转到当前硬盘的其他文件
d: #跳转到其他硬盘
cd /d e:\software #跳转到其他硬盘的其他文件夹，注意此处必须加/d参数。否则无法跳转
cd.. #跳转到上一层目录
```

4. 创建/删除目录

```cmd
md 目录名（文件夹）#创建目录
rd 目录名（文件夹）#删除目录
```

5. 删除文件

```cmd
del 文件名 #这个是专门删除文件的，不能删除文件夹
```
