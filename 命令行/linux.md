

1. where are you？

   ```
   pwd #print working directory
   ```

2. 查看当前工作目录文件

   ```
   ls
   list
   ```

3. 更改目录

   ```
   cd /地址
   cd .. #跳回上一个目录
   ```
   

4. 查看文件内容

   ```
   cat 文件名
   ```

5. 粉碎文件

   ```
   shred 文件名
   ```

6. 创建新目录

   ```
   mkdir 目录名
   ```

7. 清空屏幕

   ```
   clear
   ```

8. 创建新用户

   ```
   useradd 用户名
   ```

9. 显示用户手册

   ```
   man 命令名
   ```
